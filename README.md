# School App

학교 공지사항과 학식 정보를 한 곳에서 확인할 수 있도록 개발한 모바일 애플리케이션입니다.

사용자가 학교 홈페이지를 직접 방문하지 않아도 필요한 정보를 빠르게 확인할 수 있도록 공지사항 및 학식 메뉴를 자동으로 수집하여 제공하는 것을 목표로 개발하였습니다.

## 프로젝트 소개

기존에는 학교 공지사항과 학식 정보를 확인하기 위해 각각의 웹페이지를 방문해야 했습니다.

본 프로젝트는 학교 홈페이지의 정보를 자동으로 수집하여 모바일 환경에서 보다 쉽고 편리하게 확인할 수 있도록 개발한 애플리케이션입니다.

## 주요 기능

### 공지사항 조회

* 학교 홈페이지 공지사항 자동 수집
* 최신 공지사항 목록 제공
* 모바일 환경에 최적화된 UI 제공

### 학식 메뉴 조회

* 학식 메뉴 자동 수집
* 날짜별 메뉴 확인
* 모바일에서 간편하게 조회 가능

### 사용자 친화적 UI

* 직관적인 화면 구성
* 간편한 메뉴 이동
* 필요한 정보를 빠르게 확인 가능

## 기술 스택

Frontend

* Flutter
* Dart

Backend / Data Collection

* Python
* Web Scraping


## 담당 역할

### 학식 메뉴 스크래핑

* 학교 홈페이지에서 학식 정보 수집
* 데이터를 앱에서 활용할 수 있도록 가공

### 공지사항 스크래핑

* 공지사항 자동 수집 기능 구현
* 최신 정보 제공 기능 개발

### UI 설계

* 전체 애플리케이션 구조 설계
* 홈 화면 구성
* 메뉴 화면 설계
* 사용자 경험을 고려한 인터페이스 설계

## 프로젝트를 통해 배운 점

이 프로젝트를 통해 실제 웹 데이터를 수집하여 애플리케이션에 활용하는 경험을 할 수 있었습니다.

특히 Python 스크래핑 기능을 구현하면서 데이터를 가져오는 과정과 이를 사용자에게 보여주는 과정이 어떻게 연결되는지 이해할 수 있었습니다.

또한 Flutter를 활용하여 화면을 설계하면서 단순히 기능을 구현하는 것뿐만 아니라 사용자가 편리하게 이용할 수 있는 UI를 구성하는 것의 중요성을 배울 수 있었습니다.

## 향후 개선 사항

* 스크래핑 결과를 데이터베이스에 저장하여 로딩 속도 개선
* 데이터 캐싱 기능 추가
* 공지사항 검색 기능 추가
* 학사 일정 조회 기능 추가
* 푸시 알림 기능 추가
